# Diaspora Intelligence (MVP)

Plateforme front-end pour la collecte volontaire et conforme RGPD de données de la diaspora.
Les réponses sont stockées localement (localStorage) avec export/import JSON.

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- React Router

## Installation

```bash
npm install
```

## Lancement

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes
- Pas de backend requis : stockage local uniquement.
- Les pages Consentement et GDPR contiennent des placeholders à adapter juridiquement.
